Selamat datang di kelas FrontEnd Web Deveploper di kitaBisaKoding. kamu akan di bimbing untuk menjadi seorang FrontEnd Web Deveploper, apasih FrontEnd Web Deveploper itu?

FrontEnd Web Deveploper adalah seseorang yang bertanggung jawab membuat UI (User Interface), membuat Website yang responsive, membuat website yang user friendly, serta website yang interaktif. FrontEnd Web Deveploper fokus kepada pengalaman pengguna.

Apa saja yang akan dipelajari oleh FrontEnd Web Deveploper? kamu akan mempelajari HTML 5, CSS 3, serta JAVASCRIPT ES8, kita juga akan mempelajari cara membuat website yang responsive di semua device. Kita juga akan mempelajari beberapa FrameWork / Library untuk mempermudah dalam pengembangan web.

Yang perlu di persiapkan:

  • Niat & Semangat belajar
  • Code Editor : notepad / Atom / Bracket / VisualStudio Code / Sublime Text
    • bebas menggunakan code editor kesayangan kamu, disini akan menggunakan Visual studio code
  • Browser : Google Chrome / Mozila Firefox / Brave / Safari
    • bebas menggunakan browser pilihan kalian, disini akan menggunakan Google Chrome
