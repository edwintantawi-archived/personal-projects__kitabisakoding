Tipe Dokumen

Sebelum menuliskan kode HTML 5 sebaiknya kita mendefenisikan tipe dokumen, agar browser dapat mengetahui bahwa file ini adalah HTML 5, maka browser akan memproses dengan cara HTML 5. Dengan cara menuliskan !DOCTYPE dengan html

html

setelah kita mendefenisikan tipe dokumen, kita menuliskan tag html pada tag ini akan menjadi root/ dasar dari seluruh element HTML kita.
terus yang berada pada tag pembuka html itu apa?

Itu adalah attribut, setiap element html memiliki attribut nya masing masing, dan ada yang sama. Bagian attribut akan dijelaskan pada materi yang berbeda.
attribut yang digunakan pada tag html kali ini adalah lang atau Language (Bahasa), disini attribut itu menjelaskan bahwa konten di dalam tag html menggunakan bahasa inggris ( lang="en" ). en pada attribut lang adalah nilai attribut nilai tersebut dapat kita ganti sesuai kebutuhan. Masih banyak attribute yang dapat kita gunakan, kita akan mempelajarinya nanti.

head & body

di dalam tag root html terdapat tag head dan body kedua tag tersebut berada pada tingkatan yang sama yaitu berapa dalam tag pembungkus yang sama yaitu html. konsep ini dapat kita artikan sebagai Parent (orang tua) dan Child (anak). dimana html sebagai parent dari head dan body dan head dan body menjadi child dari html. Munggkin akan sedikit membingungkan bukan? kita akan pelajari konsep parent dan child pada materi berikutnya.

kedua tag tersebut memiliki fungsional yang berbeda.
head digunakan untuk menampung informasi mengenai halaman dokumen, mulai dari title, meta, link, dan lainnya.
sedangkan body digunakan untuk menampung isi dari dokumen html yang akan di tampilkan pada browser, berupa konten yang kita ingin tampilkan (text, gambar, video, dll).

head

Seperti pada penjelasan di atas tag head menampung informasi halaman dokumen, seperti pada kode di atas. Di dalam Tag head berisi tag meta, dan title.

Salah satu best practice dalam membuat web adalah menggunakan meta tag. meta tidak akan di tampilkan ke halaman website kita, tetapi meta tag akan memberikan informasi tambahan tentang kondisi atau isi halaman website kita kepada search engine. Ini akan berhubungan dengan yang namanya SEO (Search Engine Optimization) SEO merupakan suatu cara untuk membuat website kita berada paling atas ketika sesorang mencari pada search engine kata kunci yang tepat, yap ini akan meningkatkan jumlah pengunjung website kita. Kita tidak akan membahas lebih jauh mengenai SEO, mungkin di kesempatan lain.

kita dapat menuliskan lebih dari satu meta tag, karena setiap meta tag bisa memiliki peranan berbeda, tepatnya pada attributnya
